# 🐍 Змейка 2D (Unity)

## 🎮 Описание
Игра в стиле «Змейка» в неоновом стиле, созданная на Unity.
Основана на плавном движении и чистом игровом процессе.

## ⚙️ Особенности
- Движение по сетке
- Система подсчета очков
- Экран «Игра окончена»
- Отрастающий хвост
